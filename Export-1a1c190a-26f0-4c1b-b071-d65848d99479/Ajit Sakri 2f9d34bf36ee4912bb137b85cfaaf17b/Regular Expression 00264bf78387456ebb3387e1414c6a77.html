Regular Expression

  • Intro
    1. Regular expression are searching patterns in strings
    1. Think of regular expression as an Advance search of ctrl + f. If you learn this then you can find maybe two words at a time or find any number that starts with 9
  • Meta Character
    • Single Character
      1. \d - any number between 0 to 9
      1. \w - anything in between A-z, a-z, 0-9
      1. \s - White space
      1. DOT(.) - any character
      1. \S - Any thing that does not include white space
        1. ricewater(when applied rice\s) - I detect ricewater
        1. rice water(when applied rice\s) - i detect rice
      1. \W - Anything that does not include A-z, a-z, 0-9
        1. Any symbol like # $ %
    • Quantifiers(Specifies quantity of characters to detect)
      1. Multiplication symbol(*) - 0 or more i.e. any number character
      1. Plus symbol (+) - 1 or more characters
      1. Question Mark (?) - 0 or 1
        1. Example - The colors of the rainbow have many colours and the rainbow does not have a single colour.
        colou?rs? - colou(u repeats 0 or 1 times)rs(s repeats 0 or 1 times)
      1. {min,max} - any character in between min and max
        1. hello world how are you - {e , a} - ello world how a
      1. {n} - How many characters
    • Position
      1. ^ at beginning
      1. $ at the end
      1. \b word boundary 
        \b w+{4}\b - match any words which have 4 characters in it 
					 - Do not match any character the is a part of the word
					 - "what are chores" -> what ores❌    what✅

    To use full stop we have to use \.

    If any character is showing error then try doing \ character so that we get literal character.

  • Character classes
    1. Character classes are those that appear in between square characters which have or like operation
      1. [abc] - match a or b or c

2. [-.] - match literal -. //In character class means full stop not any character

3. (net|nonet) - match net or nonet
    1. Examples of Character classes
    1. Special characters in character classes
      1. ^ It acts as negation if it is in the start but acts as literal it it is in the middle
      1. - acts as from a to b(a-b),it works as literal - if it is at the beginning

  • Capturing Groups
    1. In previous topics we have seen how to identify different patterns in a string, In this Topic, we are going to discuss how to do operation on them like replacing some part of the string :D
    1. Putting circular bracket means we are creating a group and to access those groups we use $1 for group one or \1(we can use this in the regular expression itself) for group one

    1. $0 is not first group but the whole group
    1. .* is anything - this is a greedy
      1. we have to add ? to make .* not greedy
  • Regex in JS
    1. You create strings like this
      let s = "hello this is a normal string";
    1. You create regex like this 
      let r = /hello this is regular expression/;
    1. Converting string to regex
      new RegExp("hello i'm a string");
  • Flags→ Are a way to modify the behaviour of regular expression
    1. g - match all of them(But will not return groups 1 on wards)
    1. i - case insensitive

  • Functions in Regex

      string.function(Regular-expression)

let r = /us/;
r.test(s); //returns true
    1. .match() function

let r = /coders/;
s.match(r); // returns "coders"
